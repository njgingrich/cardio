Body Text

Normal paragraph text looks like this. By default, Cardio uses the Lato font through Google Fonts. Header fonts use Libre Baskerville, while monospace fonts use Source Code Pro. This is all done through Google Fonts with graceful fallback fonts so these are purely opional, but useful defaults.

If you like, use the lead class to give the paragraph text more emphasis.
