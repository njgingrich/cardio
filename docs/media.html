A simple but opinionated CSS framework

Media Components

Images

Cardio won't change any media styling on its own. If you wish to make an image resize inside its container, you can add the responsive class to it.

This image is 600px.

This image resizes to fit its container.

<div class="grid">
  <div class="column">
    <p>This image is 600px.</p>
    <img src="img/600x600.jpg" class="responsive"/>
  </div>
  <div class="column">
    <p>This image resizes to fit its container.</p>
    <img src="img/600x600.jpg" class="responsive"/>
  </div>
</div>

Video

Likewise, to resize a video or an embedded video (ala Youtube), you can add the responsive class to it as well.

This video is limited to 400px.

This embedded video from youtube is limited to 400px. Since it is contained within an iframe, the iframe must be wrapped within a div with the responsive class attached.

This video's container is the entire <section>, and still expands to fill 100% width.

<div class="column fixed-400">
  <p>This video is limited to 400px.</p>
  <video src="img/placeholder.mp4" controls="controls" class="responsive"></video>
</div>
<div class="column fixed-400">
  <p>
    This embedded video from youtube is limited to 400px. Since it is contained within an iframe, the iframe must be wrapped
    within a div with the responsive class attached.
  </p>
  <div class="embedded">
    <iframe width="560" height="315" src="https://www.youtube.com/embed/YSkHIv7GhOM" frameborder="0" allowfullscreen></iframe>
  </div>
</div>
<div class="column">
  <p>This video's container is the entire &lt;section&gt;, and still expands to fill 100% width.</p>
  <div class="embedded">
    <iframe width="560" height="315" src="https://www.youtube.com/embed/YSkHIv7GhOM" frameborder="0" allowfullscreen></iframe>
  </div>
</div>
