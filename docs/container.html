Containers

Container

Use a container as a responsive container but with a max-width, allowing centered content page for your layout.

This content has a max width of 1200px but is responsive below that.

Fluid Container

If you want the page to resize while still having some left and right margin centering your content, use a fluid container.

This content is centered with no max-width.

Flex Containers

Add the flex class to container or container-fluid to add a flex context to the container.
