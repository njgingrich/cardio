Cards

Base Card

Use a card to put content into a defined block.

Card Header

Text inside card body

<div class="card card-fixed">
  <header class="card-header">
    <h4>Card Header</h4>
  </header>
  <div class="card-body">
    <p>Text inside card body</p>
  </div>
</div>

Cards are fluid-width by default. By adding the card-fixed class, they are given a 400px width.

Card Body

Inside the card, you can have content, images, or buttons.

This is some text under the image.

<div class="card">
  <img src="img/1200x400.jpg">
  <div class="card-body">
    <p>This is some text under the image.</p>
  </div>
</div>

This is some text above the image.

<div class="card card-fixed">
  <div class="card-body">
    <p>This is some text above the image.</p>
  </div>
  <img src="img/1200x400.jpg">
</div>

Card Footer

The card footer usually contains links or actions with buttons. If you want buttons to be a part of the card, use the card-buttons class.
