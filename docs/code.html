Code Styling

Inline Code

Throughout the site are examples of inline code. This is done by wrapping content within <code> tags.

Code Block

Larger code examples use code blocks, which are <code> tags wrapped in <pre> tags. The blocks use highlight.js to highlight the code.

code {
  color: $color-primary-light;
  background-color: lighten($gray-light, 10%);
  border-radius: $base-border-radius;
  padding: 0.2rem 0.3rem;
}
