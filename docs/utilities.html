Cardio

A simple but opinionated CSS framework

breakpoints

Cardio ships with 4 default breakpoints defined as Sass variables. breakpoint-xs is 415px, breakpoint-sm is 769px, breakpoint-md is 1001px, and breakpoint-lg is 1401px.

The breakpoints were chosen for good defaults based on different devices - 415px for a portrait mobile device, 769px for a portrait tablet, 1001px for a desktop, and 1401px for a large desktop screen.

Hide Content

If, instead of resizing content, you want it to simply disappear between certain breakpoints, you can do so. There are a few mechanisms available to use.

  1. up-to-[size]
    • will only show content at or below breakpoint [size]
  2. at-least-[size]
    • will only show content starting at breakpoint [size]
  3. only-[size]
    • will only show content starting at breakpoint [size], until the next breakpoint.
