Column Breakpoints

By default, columns will resize until they hit a breakpoint (< 769px) and change to be full-width columns. If you wish to override this behavior, you can add a modifier to a column class of sm, md, lg, or xl. sm columns apply between 0 and 768px, md between 769px and 1000px, lg between 1001 px and 1400px, and lg > 1400px.

      <p>6 xl</p>

    <div class="col-example">
      <p>9 sm</p>
      <p>3 md</p>
      <p>2 lg</p>
      <p>6 xl</p>
    </div>
  </div>
</div>
