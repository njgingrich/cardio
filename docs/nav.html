Navbar

Define a navbar through the usual html markup, and simply add the navbar class, along with an optional color modifier class to define the color scheme for the navbar. Content can be aligned to the right by adding a second <ul> inside the navbar and giving it the right class. A second right-aligned list can be added to the nav to center the first one.
