Navs

Navbar

Define a navbar through the usual html markup, and simply add the navbar class, along with an optional color modifier class to define the color scheme for the navbar. Content can be aligned to the right by adding a second <ul> inside the navbar and giving it the right class. A second right-aligned list can be added to the nav to center the first one.

  • Item 1
  • Item 2
  • Item 3
<nav class="navbar primary">
  <ul>
    <li>
      <a href="#">Item 1</a>
    </li>
    <li>
      <a href="#">Item 2</a>
    </li>
  </ul>
  <ul class="right">
    <li>
      <a href="#">Item 3</a>
    </li>
  </ul>
</nav>

Links

Mark a link as active by adding the active class to it.

  • Link 2
  • Link 3
  • Link 4
<nav class="navbar secondary">
  <div class="brand">BigCorp</div>
  <ul>
    <li>
      <a href="#">Link 1</a>
    </li>
    <li>
      <a href="#" class="active">Link 2</a>
    </li>
    <li>
      <a href="#">Link 3</a><a href="#">Link 4</a>
    </li>
  </ul>
</nav>

Navbar Items

You can add buttons, logos, inline forms, or anything else to a navbar.

Brand Name
<nav class="navbar primary">
  <div class="brand">Brand Name</div>
  <ul>
    <li>
      <a href="#">Item Link</a>
    </li>
  </ul>
  <div class="right">
    <div class="field-inline">
      <div class="attached">
        <input class="input"/>
        <button class="outlined">Click Me</button>
      </div>
    </div>
    <button class="success">Log In</button>
  </div>
</nav>
