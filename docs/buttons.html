Cardy

A simple but opinionated CSS framework

Buttons

Overview

Use buttons with a variety of style modifiers.

Color modifiers

The available colors are primary, secondary, success, warning, and danger.

<button>Base</button>
<button class="primary">Primary</button>
<button class="secondary">Secondary</button>
<button class="success">Success</button>
<button class="warning">Warning</button>
<button class="danger">Danger</button>

Sizing

Sizing can be done with the small, large, or xlarge classes.

<button class="small">Small</button>
<button>Normal</button>
<button class="large">Large</button>
<button class="xlarge">Extra Large</button>

Outline Buttons

If you want a less imposing button, you can add the outlined class to a button.

<button>Normal</button>
<button class="primary outlined">Primary</button>
<button class="secondary outlined">Secondary</button>
<button class="success outlined">Success</button>
<button class="warning outlined">Warning</button>
<button class="danger outlined">Danger</button>

Disabled

Disabling an element should be done by attributes, not with a disabled class. This is how button[disabled] buttons look.

<button>Base</button>
<button class="primary" disabled>Primary</button>
<button class="secondary" disabled>Secondary</button>
<button class="success" disabled>Success</button>
<button class="warning" disabled>Warning</button>
<button class="danger" disabled>Danger</button>
<button>Base</button>
<button class="primary outlined" disabled>Primary</button>
<button class="secondary outlined" disabled>Secondary</button>
<button class="success outlined" disabled>Success</button>
<button class="warning outlined" disabled>Warning</button>
<button class="danger outlined" disabled>Danger</button>

Button group

You can group buttons together so they appear as a larger unit by nesting them in a grouped class. You can use any other styles you want on the buttons in combination.

<div class="grouped">
  <button>Button 1</button>
  <button>Button 2</button>
  <button>Button 3</button>
  <button class="primary">Button 4</button>
  <button class="danger" disabled>Button 5</button>
  <button>Button 6</button>
</div>
